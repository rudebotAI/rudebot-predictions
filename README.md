# Prediction Market Quant Bot v4.0 - Edge Edition

Crypto trading bot for Polymarket, Kalshi, and other prediction markets.

In PAPER mode by default -- no real money until you change the configuration.
## Files 
- `main.py` - Main FLAS file
- `setup.sh` - One-click install
- `start.sh` - Quick start launcher
- `config.yaml.example` - Copy to `config.yaml` and edit file
- `dashboard.html` - Live status page (open in browser)
